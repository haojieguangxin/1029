javascript基础之函数

函数是什么

函数是可重复执行的代码块。

函数的定义

  1. 函数声明

    function test() {
    console.log("123");  
}

  2. 函数表达式

    var test = function() {
    console.log("123");  
}

  3. 使用Function构造函数

    var test = new Function("console.log('123'); ")

函数定义的区别

函数声明提前

实例1：

test();
function test() {
    console.log("123");  
}

实例2：

test();
var test = function() {
    console.log("123");  
}

函数参数

  1. 定义一个带参数的函数（以声明式函数为例）

    function funcWithParam(a, b) {
    return a + b;
}

  2. 基本类型和引用类型

基本类型参数

    var a = 1;
    function test(a){
       a++;
    }
    test(a);
    console.log(a);

引用类型参数

    var a =[1];
    function test(a){
       a.push('2');
    }
    test(a);
    console.log(a);

    var a =[1];
    function test(a){
       a = [2];
    }
    test(a)
    console.log(a)

// 当在函数体内使用赋值操作时，系统就创建了一个变量名为a的变量。这个a是函数内部的变量，对它进行赋值当然只在函数体内起作用，外面的a还是原来的a。
