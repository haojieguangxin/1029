hexo博客搭建

1. 安装hexo

指定，就会在当前目录下创建名为的新文件夹，在新文件夹中初始化

不指定，则在当前目录下初始化

3. 初始化依赖

默认启动的端口为4000，如果需要改变访问端口，可以执行下面的命令

生成博客文件

hexo deploy

部署到github中

在部署的过程出现过下面的问题

  • 问题一：

错误信息：Deployer not found: github

解决方案：

安装插件，执行命令npm install hexo-deployer-git --save

将deploy配置中的type由github改成git

  • 问题二：

错误信息：spawn git enoen

解决方案：

配置环境变量，将git配置到path中

  • 问题三：

错误信息：Not a git repository

解决方案：

执行命令git init

  • 问题四：

错误信息： fatal: Could not read from remote repository.Please make sure you have the correct access rights.and the repository exists.

解决方案：

最开始deploy配置中的repository写成

git@github.com:yourname/yourname.github.io.git

后来改成 https://github.com/yourname/repositoryName 就可以了
